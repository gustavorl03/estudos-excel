# 📊 Excel para Análise de Dados

Este repositório documenta meus estudos e projetos práticos utilizando **Microsoft Excel** com foco em **análise de dados**.

O objetivo é desenvolver habilidades em manipulação, limpeza e análise de dados, criando soluções que auxiliem na tomada de decisão baseada em dados.

---

## 🎯 Objetivos de Aprendizado

- Dominar fórmulas e funções do Excel
- Realizar limpeza e preparação de dados
- Criar análises exploratórias
- Construir dashboards interativos
- Aplicar Excel em cenários reais de análise de dados

---

## 📚 Conteúdos Estudados

- Fórmulas básicas
- Funções lógicas
- Funções de busca (PROCV / PROCX / ÍNDICE + CORRESP)
- Manipulação e limpeza de dados
- Tabelas dinâmicas
- Gráficos e visualização de dados
- Construção de dashboards
